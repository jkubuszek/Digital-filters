 A C++ library for the design and analysis of digital signal filters (FIR, IIR). It enables easy creation of low-\/pass and high-\/pass filters, application of time windows, and generation of time response and frequency response (Bode) plots.

The library was created by {\bfseries{\mbox{[}Jakub Kubuszek/ Initials\+: JK\mbox{]}}}.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{About the project}{About the project}}\label{index_autotoc_md1}
The library is my final project for the "{}\+Object-\/oriented programming languages"{} course at AGH in Cracow.\hypertarget{index_autotoc_md2}{}\doxysubsection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{FIR Filters\+:}} Design using the window method (Low\+Pass, High\+Pass), support for custom coefficients.
\item {\bfseries{IIR Filters\+:}} Support for custom coefficients (feedforward/feedback).
\item {\bfseries{Time Windows\+:}} Available Hamming and Hanning windows.
\item {\bfseries{Analysis\+:}} Generation of data for Bode plots (magnitude and phase).
\item {\bfseries{Visualization\+:}} Built-\/in plotting functions (requires Gnuplot).
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysubsection{\texorpdfstring{Requirements}{Requirements}}\label{index_autotoc_md3}
